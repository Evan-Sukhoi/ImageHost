P.1 背景

背景与目标

前后逻辑：所有模块的前置背景与业务目标，是需求出发的原点

品牌营销中心每发布新品或促销需为多语言市场（德、法、荷、英等）同步输出各类渠道视觉物料。当前依赖人工排版，面临排版耗时、多语言版式适配难、文案校对低效、多尺寸重复劳动等痛点。本系统目标为 AI 驱动的多尺寸批量输出、多语种自适应排版、文案校对与规范内置。

目标一次性通过率>90%

排版设计耗时

P0

文案量大的线上线下物料排版耗时久，现有模型仅生成参考图，仍需人工二次调整

  • 无法直接复用 AI 生成的排版结果
  • 反复调整布局逻辑
  • 未彻底解决排版耗时问题

多语言文案版式适配

P1

多语言版本物料替换文案时排版易变形（溢出、错位），需人工重新调整

  • 不同语言文案长度差异大
  • 缺乏文案替换+自适应排版方案
  • 避免换文案→排版崩→重调的循环

多语言文案校对

P2

设计侧与社媒侧均需多语言校对，现有方案仅覆盖视频，图片文案仍依赖人工

  • 图片文案校对依赖人工
  • 缺乏全自动稳定方案
  • 需要扩展至图片场景

多尺寸适配

P0

一张图可能需适配 16 个尺寸，图片和语种一多重复劳动剧增

  • 广告图尺寸繁多
  • 多尺寸排版重复劳动量大
  • 需一键输出所有标准尺寸
项目目标

构建 AI 驱动的视觉物料生成助手，实现多尺寸批量输出、多语种自动排版、多语言文案校对、规范内置

  • 多尺寸批量输出（P0）：按渠道尺寸一键生成全部标准尺寸
  • 多语种自动排版（P1）：文案替换+自适应排版，自动调整版式
  • 多语言文案校对（P2）：自动识别拼写、语法、语义错误，覆盖图片文案
  • 规范内置：渠道尺寸、品牌素材、多语言字体内置
P.2 用户

目标用户

前后逻辑：基于业务背景识别目标用户，明确用户角色与需求

用户运营负责活动策划，提供文案与诉求，需快速获取多语种物料；品牌视觉负责物料设计与审核，需减少重复排版，聚焦创意。

用户运营

负责活动策划，提供文案与诉求，需快速获取多语种物料

  • 上传需求表格与素材
  • 选择目标渠道
  • 预览并下载物料

品牌视觉

负责物料设计与审核，需减少重复排版工作，聚焦创意

  • 提供底图与切图 PSD
  • 审核生成结果
  • 管理品牌规范与素材
P.3 功能

功能范围

前后逻辑：为核心用户提供具体的功能定义，涵盖排版生成、尺寸延展、规范管理、批次输出与文案校对

F1：多语种物料排版生成

P1

输入需求表格、底图 PSD、元素切图 PSD，输出按语言分组的合成终稿（DE/FR/NL/EN 等）。自动处理字体切换、文字长度自适应（缩字号→缩行距→告警）。

  • 输入：xlsx 文案 + 底图 PSD + 元素切图 PSD
  • 输出：按语言分组图片
  • 支持语种：DE, FR, NL, EN，可扩展
  • 自适应排版：解析坐标→合成→渲染文案
  • 字长处理：缩字号 → 缩行距 → 溢出告警

F2：多尺寸批量延展

P0

根据目标渠道自动输出所有标准尺寸，覆盖付费广告、社媒、论坛、领英、YouTube、PR与展示广告、官网等渠道。

  • 一键生成所有标准尺寸物料
  • 自动适配不同渠道需求
  • 支持 7 大渠道 30+ 种尺寸

F3：规范与资源管理

P1

内置各渠道尺寸规范，管理品牌素材库访问权限，统一多语言字体资源（支持西欧字符集）。

  • 尺寸规范内置，无需手动查阅
  • 品牌素材库（字体、Logo、色板）权限管理
  • 多语言字体统一（Ageo TTF 等）

F4：批次管理与输出

P1

支持按批次组织物料，自动按规范命名并打包下载。

  • 按批次组织物料
  • 自动命名：[渠道]_[产品]_[批次]_[宽]x[高]px_[语言].jpg
  • 支持整批次打包下载

F5：多语言文案校对 Agent

P2

对生成物料中的文案自动执行多语言校对，检测拼写、语法、语义错误，输出校对报告并支持自动修正与手动触发。

  • 覆盖图片文案场景
  • 检测项：拼写、语法、语义
  • 输出校对报告，标注问题位置与建议
  • 支持自动触发与手动发起校对
  • 多语种支持：DE, FR, NL, EN，可扩展

系统核心功能包括

  • 多语种物料排版生成（F1）、多尺寸批量延展（F2）、规范与资源管理（F3）、批次管理与输出（F4）、多语言文案校对 Agent（F5）。

P.4 流程

用户流程

前后逻辑：串联功能的端到端操作路径，展示从输入到输出的完整流程

市场/运营上传需求表格、底图 PSD、元素切图 PSD，选择目标渠道；系统自动解析文案、读取坐标、合成底图与元素、渲染文案、执行排版合规检测与文案校对；用户按语言/渠道/尺寸预览，查看校对报告并支持单张调整重生成；最后按规范打包下载。

操作步骤

仅需 5 步完成一批次：上传 → 选择渠道 → 系统处理 → 预览与校对 → 导出

  • 1. 上传需求表格（xlsx）+ 底图 PSD + 元素切图 PSD
  • 2. 选择目标渠道（可多选）
  • 3. 系统自动处理：解析→排版→自适应渲染→校对
  • 4. 预览：按语言/渠道/尺寸分组查看，支持单张调整重生成
  • 5. 批量导出：按规范命名打包下载
系统自动处理细节

解析 xlsx 提取文案，读取底图 Artboard 获取坐标规则，合成底图与元素，渲染文案并自适应排版，完成后触发排版合规检测与多语言校对。

  • 解析文案与坐标规则
  • 合成底图 + 元素切图
  • 渲染文案并自适应排版
  • 合规检测（溢出、安全区、分辨率）
  • 多语言校对（P2 自动触发）

P.8 技术

技术方向

前后逻辑：技术实现方案，涵盖输入解析、图层合成、排版规则、字体管理、输出格式

采用 xlsx 直接解析文案，psd-tools 读取 Artboard 图层坐标；从底图 PSD 导出背景+元素合成图，用 Pillow 渲染文案；排版规则基于历史 PSD 提取坐标，跨语言坐标高度一致；字体管理使用 Ageo TTF 按字重加载；输出 JPG 格式。

关键技术点

输入解析、图层合成、排版规则、字体管理与输出格式

  • 输入解析：xlsx 直接解析文案，psd-tools 读取 Artboard 坐标结构
  • 图层合成：导出底图+元素合成图，叠加切图，Pillow 渲染文案
  • 排版规则：历史 PSD 提取坐标规则，新任务直接套用
  • 字体管理：项目目录 Ageo TTF，按字重加载，支持西欧字符集
  • 输出格式：JPG，分辨率按渠道规范配置
