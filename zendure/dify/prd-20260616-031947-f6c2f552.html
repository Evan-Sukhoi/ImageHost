P.1 背景

背景与痛点

前后逻辑：作为产品叙事的起点，后续功能均围绕解决这些痛点展开

品牌营销中心在新产品发布或促销时，需为德语、法语、荷兰语、英语等多语言市场，同步输出广告、社媒、论坛、领英、官网等渠道视觉物料。当前流程依赖设计师手动排版，存在排版耗时、多语言版式适配困难、多语言文案校对人工、多尺寸适配重复劳动四大痛点。

排版设计耗时（P0）

P0

文案量大的线上/线下物料排版耗时久，需反复调整布局逻辑。现有 GPT-Image2 仅生成参考效果，仍需人工二次调整，未彻底解决排版耗时问题。

  • 排版耗时久，需反复调整布局
  • 现有图像模型仅提供参考底图，无法直接复用
  • 仍未解决排版自动化问题

多语言文案版式适配（P1）

P1

多语言版本物料需逐语言替换文案，不同语言文案长度差异导致排版变形（文字溢出、版式错位），需人工重新调整排版，缺乏自动化方案。

  • 多语言文案替换后排版崩坏
  • 文案长度差异导致错位、溢出
  • 无「文案替换+自适应排版」自动化方案

多语言文案校对（P2）

P2

设计侧和社媒侧的图片文案均需多语言校对，缺乏稳定、全自动的校对方案。现有 Gemini 方案仅覆盖视频文案，图片文案校对仍依赖人工。

  • 图片文案校对依赖人工
  • 现有自动校对仅覆盖视频，未覆盖图片
  • 需构建全自动多语言校对 Agent

多尺寸适配（P0）

P0

一张图可能需适配16个尺寸，图片和语种一多，重复性劳动剧增。

  • 一张图可能需适配16个尺寸
  • 图片和语种增多时重复劳动剧增
  • 亟需批量自动延展方案

前后逻辑：明确产品解决的核心问题及期望达成的量化结果

构建 AI 驱动的视觉物料生成助手，实现多尺寸批量输出、多语种自动排版、多语言文案校对，并将渠道尺寸规范、品牌素材、多语言字体资源内置于系统中。成功指标为视觉物料一次性通过率大于90%。

P1

将各渠道尺寸规范、品牌素材、多语言字体资源统一内置于系统。

P.4 核心功能

多语种物料排版生成（F1）

前后逻辑：接收上传的需求表格、底图PSD和元素切图后，执行核心的排版生成逻辑

解析需求表格提取各语言文案，从底图PSD读取Artboard坐标规则，合成底图与元素切图后渲染文案，并自动处理字体切换、文字长度自适应。

输入要求

系统需要三类输入文件以启动生成。

  • 需求表格（xlsx）：各语言文案、目标尺寸、特殊要求
  • 底图 PSD：背景与装饰元素，不含文字层，每个 Artboard 对应一种尺寸
  • 元素切图 PSD：新产品图、图标等视觉素材

输出结果

按语言分组的合成终稿。

  • 支持语言：德语（DE）、法语（FR）、荷兰语（NL）、英语（EN），可扩展
  • 输出按语言分组，如 DE/FR/NL/EN 文件夹

处理逻辑

从输入解析到渲染输出的自动化流程。

  • 解析需求 xlsx，提取各语言文案
  • 从底图 PSD Artboard 读取元素坐标，套用历史排版规则
  • 合成底图 + 元素切图，按坐标规则用 Pillow 渲染各语言文案
  • 自动处理字体切换、文字长度自适应（缩字号 → 缩行距 → 告警）
P.5 核心功能

多尺寸批量延展（F2）

前后逻辑：在单个尺寸底图生成后，按渠道规则批量延展其他所需尺寸

根据目标渠道自动输出对应所有标准尺寸，覆盖付费广告、社交媒体、论坛、领英、YouTube、PR展示广告、官网等七大渠道，每个渠道均有多组固定尺寸。

渠道与尺寸覆盖

系统内置七大渠道的完整尺寸规范。

  • 付费广告：14种尺寸（横版、方形、竖版、Stories、Display等）
  • 社交媒体：横版帖子(1200x628)、竖版帖子(1350x1080)
  • 论坛：标准横幅(1500x930)、宽屏横幅(2058x1020)
  • 领英：封面/帖子(2880x1620)、竖版(880x1620)
  • YouTube：缩略图(1200x720)、Shorts封面(1080x1920)
  • PR展示广告：涵盖小图到全屏展示等12种尺寸
  • 官网PC端：Hero Banner(1920x800)、内容区(1920x600)等；移动端以750x720等为标准输出填充图

P.9 流程

用户操作流程

前后逻辑：描述从输入到最终交付的端到端步骤

市场/运营人员上传需求文件并选择渠道后，系统自动解析、排版、校对，用户预览审核后可批量打包下载。

输入与配置

用户上传需求表格、底图PSD、元素切图PSD，并选择目标渠道。

  • 上传需求表格（xlsx）：各语言文案 + 目标尺寸 + 特殊要求
  • 上传底图 PSD（含各尺寸 Artboard，不含文字层）
  • 上传元素切图 PSD（产品图、图标等）
  • 选择目标渠道（可多选）
系统自动处理

系统解析 xlsx 提取文案，读取底图 PSD Artboard 获取排版规则，合成底图与元素切图，渲染多语言文案并执行自适应排版、合规检测和多语言校对。

  • 解析 xlsx → 提取各语言文案
  • 读取底图 PSD Artboard → 获取各尺寸排版规则
  • 合成底图 + 元素切图
  • 按坐标规则渲染各语言文案（自适应排版）
  • 排版合规检测（溢出、安全区、分辨率）
  • 多语言文案校对（P2，自动触发）
预览与审核

用户可按语言/渠道/尺寸分组预览物料，查看校对报告，并支持单张调整重新生成。

  • 按语言/渠道/尺寸分组预览
  • 查看校对报告（文案问题标注）
  • 支持单张调整后重新生成
批量导出

系统按规范命名打包，用户一键下载整批次物料。

  • 按规范命名打包下载
  • 文件夹按渠道/语言组织
